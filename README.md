# MadrugaCRUD

Este projeto possui um sistema feito em MVC VB.NET com vue.js com as 4 operações básicas CRUD.

+ Menu
    + Tela de cadastro de cliente
    + Tela de cadastro de estado
    + Tela de cadastro de cidade
